# 프로젝트 목표 (Project Goals)

## 1. 개요
일별 최대전력(MW) 시계열을 외생변수(독립변수)와 함께 모델링하여
**ARIMAX / SARIMAX** 기반 예측 모델을 구축하고, 2025년 실측치로
모델의 일반화 성능을 검증한다.

## 2. 데이터

### 2.1 종속변수 (Target)
- **변수**: 일별 최대전력 (MW)
- **출처**: 한국전력거래소(KPX) EPSIS — 일별 최대전력 기준 전력수급 정보
  - EPSIS: https://epsis.kpx.or.kr/epsisnew/selectEkgeEpsMepChart.do?menuId=030100
  - 공공데이터포털 API: https://www.data.go.kr/data/15150737/fileData.do
- **기간**: 2010-01-01 ~ 2025-12-31 (16년, 5,844일)
- **세부 메타**: `docs/dependent_var.md`
- **기간 확정 근거**: `docs/research_log/2026-05-25_종속변수_기간확정_2010-2025.md`

### 2.2 독립변수 (Exogenous Features) — **수집 예정**
종속변수 기간(2010–2025)에 정합되는 일 단위 외생변수를 수집한다.
후보 카테고리:

| 카테고리 | 예시 |
|----------|------|
| 기상 | 일평균/최고/최저 기온, 체감온도(HDD/CDD), 습도, 일사량, 풍속 |
| 달력 | 요일, 월, 공휴일, 연휴 길이, 명절(설/추석) 더미 |
| 경제·산업 | 산업생산지수, 제조업 가동률 등 (월별 → 일별 분배 필요) |
| 기타 | COVID-19 등 구조 변화 더미 |

- 변수 후보·출처는 `docs/independent_var.md`에 정리 예정.
- 모든 독립변수는 2010-01-01 ~ 2025-12-31 일 단위로 정렬·정합한다.

## 3. 모델링 계획

### 3.1 분할
- **학습 구간**: 2010-01-01 ~ 2024-12-31 (15년, 5,479일)
- **평가 구간**: 2025-01-01 ~ 2025-12-31 (1년, 365일) — 홀드아웃
- 학습 구간 내부에서 시계열 교차검증(rolling/expanding window)으로
  하이퍼파라미터 선택.

### 3.2 모델
- 기본: **ARIMAX** — 외생변수 포함 ARIMA.
- 확장: **SARIMAX** — 주간/연간 계절성 + 외생변수.
- 정상성 점검(ADF/KPSS), 차분 차수 결정, ACF/PACF 기반 차수 후보 선정,
  AIC/BIC + 잔차 진단(Ljung–Box 등)으로 최종 모델 확정.

### 3.3 평가 지표
- 점예측: MAE, RMSE, MAPE, sMAPE
- 구간예측: 95% 예측구간의 커버리지·평균 폭
- 베이스라인 비교: Naive(전일), Seasonal Naive(전주/전년 동일), SARIMA(외생변수 없음)

## 4. 산출물
- 전처리 데이터셋 (종속변수 + 정합된 독립변수, 일 단위)
- 학습된 ARIMAX / SARIMAX 모델과 진단 리포트
- 2025년 예측 vs 실측 비교 결과 및 오차 분석
- 최종 보고서 / 발표 자료
