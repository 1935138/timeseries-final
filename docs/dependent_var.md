# 종속변수 (Target) — 일별 최대전력 메타

## 0. 문서 목적

본 문서는 ARIMAX/SARIMAX의 **종속변수**인 일별 최대전력(`peak_mw`)에 대한
정의·출처·가공 방안·검증 결과를 단일 메타로 정리한다. 독립변수 메타는
`docs/independent_var.md`, 의사결정 이력은 `docs/research_log/`에 있다.

- 정합 기간: **2010-01-01 ~ 2025-12-31 (5,844일, 결측 0)**
- 최종 산출물: `data/processed/peak_mw_2010_2025.csv` (5,844행, `tm, peak_mw`)
- 단위: **MW (메가와트)** · 시간대: **KST**
- 상위 계획: `docs/project_goals.md`

## 1. 정의

| 항목 | 값 |
|---|---|
| 변수명 | `peak_mw` |
| 한국명 | 일별 최대전력 |
| 정의 | 해당 일자(KST 00:00~23:59) 내 **1시간 평균 전력수요의 최댓값** |
| 단위 | MW |
| 시간 정렬 | KST 일(day) 단위, `tm` 컬럼 (`YYYY-MM-DD`) |
| 관측 기준 | 한국전력거래소(KPX) 계통 운영 실적 — 전국 통합 계통 |
| 결측 | 0건 (2010-01-01 ~ 2025-12-31 전 구간 완전) |
| 사후 개정 | KPX 실적치 — 통상 익월 확정 후 안정. vintage 이슈 미미 |

원본의 "최대전력기준일시" 컬럼에 따르면 최댓값이 발생한 정확한 시각이
함께 기록되며, 일별 최대전력은 그 시각의 1시간 평균값이다.

## 3. 원본 컬럼 명세 (11개)

원본 CSV는 일별 전력수급의 다중 지표를 포함하지만, **본 프로젝트는
`최대전력(MW)` 단일 컬럼만** 종속변수로 사용한다.

| 원본 컬럼 | 단위 | 사용 | 비고 |
|---|---|---|---|
| 년 | int | ✓ (조합) | `tm`으로 통합 |
| 월 | int | ✓ (조합) | `tm`으로 통합 |
| 일 | int | ✓ (조합) | `tm`으로 통합 |
| 설비용량(MW) | MW | ✗ | 발전설비 총용량 — 수요와 무관 |
| 공급능력(MW) | MW | ✗ | 공급측 변수, 수요 예측 입력 부적합 |
| **최대전력(MW)** | **MW** | **✓ → `peak_mw`** | **본 프로젝트 종속변수** |
| 최소전력(MW) | MW | ✗ | 별도 모델 대상 (본 프로젝트 범위 외) |
| 공급예비력(MW) | MW | ✗ | 공급능력 − 최대전력 = 사후 산출치, 역인과 |
| 공급예비율(%) | % | ✗ | 동상 — 독립변수 후보에서도 제외 (§ `independent_var.md` 5.2) |
| 최대전력기준일시 | datetime | ✗ | 일별 시간 정보 — Phase 1에서는 미사용 |
| 최소전력기준일시 | datetime | ✗ | 동상 |

## 4. 가공 절차

1. 원본 인코딩 `cp949` → 출력 `utf-8` (BOM 없음).
2. `2010 ≤ year ≤ 2025` 필터링 (사유는 §6 기간 결정).
3. `(년, 월, 일)` 기준 중복 제거 — 총 13건 (모두 행 전체 값 동일):
   - 2010-03-24, 2010-04-21, 2010-08-27 (3건)
   - 2023-04-01 ~ 2023-04-10 (10건)
4. `(년, 월, 일)` → `tm` (`YYYY-MM-DD`) 단일 컬럼으로 통합.
5. `최대전력(MW)` → `peak_mw`로 리네임, 외 컬럼 drop.
6. `tm` 오름차순 정렬 후 저장.

**최종 스키마**: `tm` (date), `peak_mw` (int, MW) — 2열, 5,844행.

## 5. 검증 결과

### 5.2 기초 통계량

| 통계 | 값 (MW) |
|---|---|
| min | 38,764 (2010-09-22) |
| mean | 68,079 |
| std | 9,097 |
| max | **97,115 (2024-08-20)** — 역대 최고 |

연도별 평균이 2010년 **59,192 MW** → 2024년 **73,235 MW**로 약 **23.7% 상승**
(CAGR ≈ 1.53%) — 장기 추세 동인(인구·산업생산) 외생변수의 필요성 근거.

### 5.3 계절성·이벤트

- **월별 평균 (MW)**: 1월 73,703 / 8월 74,593 / 12월 73,509 — **여름·겨울
  쌍봉**, 환절기(4·5·10월) 저점 약 61,000~63,000.
- **여름 피크 일자 top3** (모두 2024–2025년 폭염): 2024-08-20 (97,115),
  2025-08-25 (95,951), 2025-07-08 (95,675).
- **최저 일자 top3**: 모두 9월 말 추석 연휴 직후 (2010-09-22, 2010-09-23,
  2012-09-30) — 휴일·기상 변수의 결합 효과.

## 6. 기간 결정 경위 (요약)

원본은 1993–2026이지만 다음 사유로 **2010–2025년 16년**만 사용:

- **1993–2006**: 결측 다수·부분 데이터, 시계열 학습 불가.
- **2007**: 완전 1년이나 2008·2009 결측으로 연속성 단절.
- **2008·2009**: 실제 일자 누락(각 3·4일), 보간 시 노이즈.
- **2026**: 5월 23일까지 진행 중 — 1년 단위 일관성 위해 제외.
