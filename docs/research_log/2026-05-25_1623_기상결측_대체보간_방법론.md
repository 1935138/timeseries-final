**bias 보정 대체 산식**
대표 관측소 `A`의 결측일 `t`에 대해 백업 관측소 `B`의 같은 날 값을 평균
차이를 보정해 사용한다.

```
x̂_A(t) = x_B(t) + (μ_A,train - μ_B,train)
```
- `μ_A,train`, `μ_B,train`: **학습 구간 2010-01-01 ~ 2024-12-31** 동안의
  같은 컬럼 평균. 양 관측소 모두에서 값이 있는 날만 사용.
- 학습 구간 통계만 사용해 **미래 정보 누설 방지** (2025 홀드아웃 보장).
- 보정량은 컬럼별로 별도 계산.

**컬럼별 적용 여부**

| 컬럼군 | bias 보정 대체 | 사유 |
|---|---|---|
| `ta_avg, ta_max, ta_min` | ✅ | 인접 관측소 간 상관 높고 bias 일정 |
| `hm_avg` | ✅ | 비교적 안정 |
| `ws_avg` | ⚠️ | 지형 영향 큼. 대체하되 검증 필요 |
| `rn_day` | ❌ | 강수는 국지성이 강해 인접소 대체 부적절 → 결측은 결측으로 둠 |
| `ss_day, si_day` | ❌ | 일조·일사 미관측 관측소(천안·울산·창원)가 있어 별도 처리 (가중 평균 산출 시 누락 관측소 제외 후 재정규화) |

**잔여 결측 처리** — pandas `interpolate(method='time', limit=3)`,
연속 3일까지만 보간(그 이상은 분석에서 제외 검토).

**출처(Provenance) 추적** — 각 컬럼별 `<컬럼>_src` 컬럼 추가:

| 값 | 의미 |
|---|---|
| `raw` | 원본 관측값 |
| `backup_<stn번호>` | 백업 관측소 (예: `backup_119`) |
| `interp` | 시점 선형 보간 |
| (NaN) | 모든 단계 후에도 결측 |

이를 통해 사후 분석에서 "보간 비율" 같은 품질 지표 계산 가능.

### 산출물 / 적용 범위 (예정)

  - 컬럼: 기존 raw 컬럼 + `<컬럼>_src`

  생성의 직전 입력.

### 검증 계획
대체·보간 적용 후 다음을 점검:
1. **결측 잔존 수**: 컬럼별로 0에 가까운지 (강수 제외)
2. **출처 분포**: 각 컬럼·관측소별 `raw / backup_* / interp` 비율 표
3. **시계열 연속성**: 대체된 일자 ↔ 인접일 값 차이가 비정상적으로 크지 않은지
4. **분포 보존**: 대체 전후 컬럼별 분포(min/max/mean/std) 차이가 작은지
