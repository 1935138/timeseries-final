# Research Log

## 2026-05-25 — 냉방도일(CDD) · 난방도일(HDD) 변수 정의 및 채택안

### 목적

전력-기온 관계의 **V자 비선형성**을 ARIMAX의 선형 회귀 항으로 잡기 위해
기온 자체 대신 CDD/HDD로 변환한다. 변수 정의·기준온도·이동평균 차수를
국내외 표준 문헌 비교를 통해 사전에 확정한다.

### 핵심 원리

전력 ↔ 기온은 18°C 부근을 꼭짓점으로 V자 (난방·냉방 양방향에서 증가).
기온을 그대로 선형회귀에 넣으면 양쪽 효과가 상쇄돼 계수가 0에 가깝게 추정됨.
**기준온도 b에서 양쪽으로 쪼개 양의 값으로** 만들어 별개 계수로 추정:

```
CDD_b(t) = max(T_avg(t) - b, 0)
HDD_b(t) = max(b - T_avg(t), 0)
```

### 결정 사항

| 항목 | 채택 | 근거 |
|---|---|---|
| 기준온도 b | **18°C** | EIA(미국)·EU JRC·한전 전력연구원·에너지경제연구원 모두 18°C 표준 |
| 입력 기온 | **`ta_avg` (인구가중 전국평균 일평균)** | 일평균 기반이 학계 압도적 표준 (Engle 1986 이래) |
| lag/ma 처리 | **ma3 + ma7 동시** | 건물 열관성·누적 부하 반영 (KEPRI·KPX 표준) |

| 변수 | 산식 | 단위 | 근거 문헌 |
|---|---|---|---|
| `cdd_18` | `max(ta_avg - 18, 0)` | °C·일 | EIA, KEPRI 부하예측 표준방법론(2020) |

### 문헌 비교 — 기준온도

| 출처 | base | 비고 |
|---|---|---|
| 미국 EIA·NOAA·ASHRAE | 65°F = 18.3°C | 영미권 표준 |
| EU JRC, Eurostat | 18°C | 범유럽 보고서 |
| 영국 Met Office | 15.5°C | 단열·복사난방 차이 |
| 일본 CRIEPI | CDD 24°C, HDD 14°C | 비대칭 base |
| **한전 전력연구원(KEPRI)** | **18°C** | 한국 전력 부하예측 표준 |
| **에너지경제연구원(KEEI)** | **18°C** | 가정·산업 모두 |
| Engle et al.(1986, JASA) | 데이터에서 추정 | semiparametric 회귀 (학술적 정통) |

→ 한국 전력 도메인은 **18°C 단일값**이 사실상 표준. 본 프로젝트도 18°C 채택.

### 문헌 비교 — lag/ma 처리

| 방식 | 대표 문헌 | 채택 여부 |
|---|---|---|
| (A) 단일 일별만 | Engle et al.(1986) | 열관성 미반영 → 단독으로는 부족 |
| (B) lag 1·2·3일 | Sailor & Muñoz(1997), Pardo(2002), Hor(2005) | ma와 정보 중복 → ma 채택 시 제외 |
| (C) **이동평균(MA)** | Bessec(2008) MA-7, Apadula(2012) MA-3, Mirasgedis(2006) MA-7 | **채택** — 누적 부하 효과 반영 |
| (D) EWMA (지수가중) | Henley(1998), KEPRI EWMA(λ=0.7) | Phase 2 검토 — hyperparameter 튜닝 필요 |
| (E) 비대칭 + 임계 더미 | KEPRI(2019), 이상호·박종배(2017) | 본 모델은 §3.1 `heat_th_*`·`cold_th_*`가 이 역할 |

### 가공 주의사항

**ma 산출 시 미래 정보 누설**:
- pandas `rolling(window).mean()` 은 기본적으로 **과거 포함 현재까지** ⇒ `cdd_18_ma3[t] = mean(cdd_18[t-2:t+1])`
- 이는 **현재 정보를 포함**하므로 forecast 시 t 시점에서 가용 (문제 없음)
- 단, `min_periods=window` 강제 (시계열 초반 NaN 유지) 후 backfill 처리

**시계열 초반 결측**:
- ma3: 2010-01-01 ~ 2010-01-02 (2일) NaN → 첫 가용 값으로 backfill
- ma7: 2010-01-01 ~ 2010-01-06 (6일) NaN → 동일 처리
- 학습 시작 시점이므로 정보 누설 아님
