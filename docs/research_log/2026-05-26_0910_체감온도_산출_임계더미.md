# Research Log

## 2026-05-26 — 시간별 체감온도 산출 + 폭염·한파 임계 더미

### 산출물

| 파일 | 행 수 | 설명 |
|---|---|---|
| `data/processed/feels_like_daily_2010_2025.csv` | 5,844 | `feels_like_max/min`, `ta_hourly_max/min` |
| `data/processed/extreme_flags_2010_2025.csv` | 5,844 | 4개 임계 더미 |

### 처리 흐름

1. 16개 imputed 시간자료 (`asos_hourly_imputed/`) 로드
2. 연도별 시·도 인구 가중치(`sido_weights_2010_2025.csv`) merge
3. **시간별 인구가중 전국평균** (TA, HM, WS) 산출 → 140,256 시각
4. 월에 따른 **여름/겨울 공식 분기**로 시간별 PT 산출
5. 일 단위 max/min 집계 → 5,844행

### 체감온도 적용 (research_log 2026-05-25_2139에 명시)

- 5/1 ~ 9/30: 여름 공식 (Stull Tw + 기상청 PT_summer)
- 10/1 ~ 4/30: 겨울 공식 (NWS Wind Chill, 적용 조건 T≤10°C AND V≥1.3m/s)
- 연중 단일 컬럼 `feels_like`

### 검증

| 지표 | 값 |
|---|---|
| 시간 PT 전체 범위 | -22.2 ~ 35.7°C |
| 6-8월 PT 최대 | **35.70°C** (실측 ta_max 37.12°C) |
| 12-2월 PT 최저 | **-22.17°C** (실측 ta_min -16.33°C, 풍속 가중 -6°C) |
| 폭염일 2018-08-01 | feels_like_max **35.70°C**, min 28.59°C |
| 한파일 2018-01-26 | feels_like_min **-21.77°C**, max -13.20°C |

→ 풍속 가중으로 한파일 체감이 실측보다 약 7°C 낮음. NWS 공식 효과 정상.

산출 시간: 1.2초.

### 임계 더미 (4개) — 기상청 공식 발효기준 정합

| 변수 | 정의 | 기상청 조항 |
|---|---|---|
| `heat_feels_th_2day_adv` | `feels_like_max ≥ 33°C` 2일 연속 | 폭염주의보 ① |
| `heat_feels_th_2day_wrn` | `feels_like_max ≥ 35°C` 2일 연속 | 폭염경보 ① |
| `cold_th_2day_adv` | `ta_hourly_min ≤ -12°C` 2일 연속, **10~4월 한정** | 한파주의보 ② |
| `cold_th_2day_wrn` | `ta_hourly_min ≤ -15°C` 2일 연속, **10~4월 한정** | 한파경보 ② |

**한파에 PT가 아닌 `ta_hourly_min` 사용**: 기상청 한파 ②조건은 "아침 최저
**기온**" 기준 — PT가 아닌 실측 기온. 정합성 유지.

### 제외된 조항·변수

| 항목 | 제외 사유 |
|---|---|
| 폭염·한파 **②/③조건** ("급격한 상승/광범위한 피해") | 정성적 — 수식 불가 |
| 한파 **①조건** (전일대비 -10/-15°C 급강하 + **평년값 -3°C 이상 낮음** + ≤3°C) | **평년값** (1991-2020 30년 평년 `ta_min`) 별도 산출이 추가 작업이고, 본 모델은 한파 ②(절대 기온)만으로 한파 부하 점프 대부분 포착 가능. ②와 OR 결합 시 결과 변동 미미. |
| 1일 단순 더미 (`heat_th_advisory` 등) | 공식 발효기준이 모두 "2일 이상 지속" — 1일 단순 더미는 공식 비정합. 본 모델 §2.3 `cdd_18`·`cdd_18_ma3` 가 1일 효과를 이미 흡수. |
| 공식 특보 (기상청 통보문 기반) | (A) 임계 더미와 정보 중첩 90%+ 예상, 시·군·구 집계 규칙 결정 비용 ↑ → Phase 2 검토 |
