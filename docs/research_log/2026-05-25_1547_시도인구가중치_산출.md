## 2026-05-25 — 17개 시·도 주민등록인구 수집 및 16개 단위 가중치 산출

> KOSIS `DT_1YL20651E`에서 2010–2025년 시·도 주민등록인구를 수집하고,
> 세종을 충남에 합산해 **16개 시·도 × 16년 = 256개 가중치**를 산출.
> 시·도 합 = 전국, 연도별 가중치 합 = 1.0 검증 완료.

### 대상
**기상 외생변수의 공간 가중에 사용할 시·도 인구 가중치를 산출한다.**
방법론·세종 통합 결정의 근거는
`docs/research_log/2026-05-25_1537_대표관측소_인구가중평균.md` 참조.

### 최종 결정

**KOSIS 통계표 `DT_1YL20651E` 사용 — 연별, 1992년~ 가용**

OpenAPI 호출 사양:
- 엔드포인트: `https://kosis.kr/openapi/Param/statisticsParameterData.do`
- 파라미터:
  - `orgId=101` (행정안전부), `tblId=DT_1YL20651E`
  - `itmId=T20` (계만; T21=남, T22=여 미사용)
  - `objL1`: 전국(00) + 17개 시·도 코드를 `+`로 연결
    (`00+11+26+27+28+29+30+31+36+41+51+43+44+52+46+47+48+50+`)
  - `prdSe=Y`, `startPrdDe=2010`, `endPrdDe=2025`
  - `format=json`, `jsonVD=Y`

**파싱·필터**
- 응답 286행 → `len(C1)==2`로 시·도 단위(전국 포함 17개 코드)만 추출
- 전국(`C1=00`)은 검증용으로 분리, 본 데이터에서는 제외
- `itmId=T20`만 요청 → ITM_NM 후필터 불필요

**세종 통합 + 가중치 산식**
```
# 세종(36) 인구를 충남(44)에 합산
df.loc[df.sido_code == "36", "sido_code"] = "44"
df["sido_name"] = "충청남도(세종통합)"
df = df.groupby(["year", "sido_code"])["population"].sum()

weight_i(y) = pop_i(y) / Σ_j pop_j(y),  Σ_i weight_i(y) = 1
```
